# Sweet Treats Management System

## Setup
1. Install NetBeans and MySQL
2. Open MySQL Workbench and create a database called sweet_treats
3. Run sweet_treats_sql.sql then alter_receipts.sql
4. Open the ClassicTreats folder in NetBeans
5. In LoginSystem, Dashboard and SignUpSystem change your_password_here to your MySQL password
6. Run LoginSystem.java

## Testing
- First sign up as a Customer to test the system
- To test other roles like Admin and Manager check the users table in MySQL Workbench for existing accounts
